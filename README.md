# Hastebin VS Code Extension

Share code to Hastebin directly from VS Code. `(hastebin.ianhon.com)`

## Commands

Open the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and type:

- **Hastebin: Share Selection** - Share selected text
- **Hastebin: Share File** - Share entire file
- **Hastebin: Share Multiple Files** - Select and share multiple files

The link is copied to your clipboard automatically.

## Settings

- `hastebin.apiUrl` - API endpoint (default: https://backend.ianhon.com/hastebin)
- `hastebin.siteUrl` - Website URL (default: https://hastebin.ianhon.com)
- `hastebin.openInBrowser` - Open paste in browser after creation (default: true)
- `hastebin.commentsEnabled` - Enable comments on pastes (default: true)
